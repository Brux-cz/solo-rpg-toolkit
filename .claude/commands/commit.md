Podivej se na vsechny zmeny (git diff + git status) a vytvor commit podle pravidel v CLAUDE.md:

1. Analyzuj zmeny — co se zmenilo a proc
2. Zvol spravny typ (feat/fix/refactor/docs/style/chore) a oblast
3. Napis commit message cesky ve formatu `typ(oblast): popis`
4. Commitni

Pokud je vic nezavislych zmen, rozloz je do vice commitu.
